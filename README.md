# Roberto Otarola - Developer Portfolio

Bienvenido al repositorio de mi portafolio profesional, creado para ser desplegado en **GitHub Pages**. Este proyecto centraliza mi trayectoria técnica, proyectos destacados y cuenta con un caso de estudio detallado que documenta el proceso de optimización del proyecto *Alke Web CV*.

## 🚀 Tecnologías Utilizadas
- **HTML5:** Estructura semántica base.
- **CSS:** Integración del kit de marca profesional *"Earthy & Trustworthy"* (`#E9C46A`, `#264653`, `#2A9D8F`).
- **JavaScript:** Inyección dinámica de componentes comunes (footer centralizado) para facilitar el mantenimiento.
- **Bootstrap 5:** Sistema de grillas, responsividad adaptativa y componentes de interfaz asíncronos.
- **Git / GitHub Pages:** Para el control de versiones local e integración de despliegue continuo en la web.

## 📋 Arquitectura del Proyecto
- `index.html`: *Landing page* principal que consolida el Extracto Profesional, datos de contacto, titular técnico (SEO) y las tarjetas de mis tres proyectos estelares (*Alke Wallet*, *PokeParty Optimizer* y *Alke Web CV*).
- `case-study.html`: Documento de diseño embebido detallando exhaustivamente el requerimiento de evaluación (Puntos 1 a 8), justificando decisiones arquitectónicas y aprendizajes durante la refactorización de *Alke Web CV*.
- `assets/`: Contiene `css/styles.css` con la identidad visual centralizada y `img/` con vistas previas hiperrealistas de las plataformas web trabajadas.

> 🎓 **Nota:** Este portafolio fue elaborado como entregable de los módulos de empleabilidad correspondientes a las evaluaciones **ABP #8** y **ABP #9** del Bootcamp de Desarrollo Fullstack Python Trainee.
